Муковисцидоз - это наследственное аутосомно-рецессивное заболевание, вызванное нарушением функции белка CFTR, который регулирует транспорт ионов хлора через клеточные мембраны. Это приводит к образованию густой и вязкой слизи, поражающей лёгкие, поджелудочную железу и другие органы. Основные проявления включают хронические инфекции дыхательных путей и нарушения пищеварения.

1. CFTR (Cystic Fibrosis Transmembrane Conductance Regulator) <- главный ген, мутации в нём вызывают заболевание
